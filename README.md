# WildWhere AI

Live project: https://wildwhere-ai.vercel.app

WildWhere AI is a portfolio-grade wildlife intelligence project built with real public biodiversity, park, weather, and visitation datasets. The current version includes a validated data pipeline, SQLite warehouse, analytics layer, machine-learning workflow, prediction-safety gate, and public Vercel preview.

The public preview is the primary way to review the project. It avoids unsupported park/species predictions and clearly labels outputs as historical reported-observation patterns, not confirmed wildlife presence or real-time tracking.

The full project source lives in `wildwhere-ai/`.

The earlier lightweight browser demo remains in `docs/`, but the Vercel project above is the current portfolio-ready build.
